## Usage Examples

```typescript
// Simple translation
{t('common.save')}

// With fallback
{t('some.key', 'Default text')}

// With variables
{t('validation.minLength', { min: 6 })}

// In attributes
<Input placeholder={t('auth.email')} />
<Button>{t('common.submit')}</Button>
```
